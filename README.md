# 2026년 재취업지원서비스 시행지원 홈페이지

케이잡스(K·JOBS) 수행기관 관점의 재취업지원서비스 시행지원 홈페이지 원본 저장소입니다.

## 범위

- 기업컨설팅
- 커리어플래닝서비스
- 진행 절차
- K·JOBS 지원 방식
- 문의

## 디자인 기준

이 사이트의 기본 톤앤매너는 커리어플래닝서비스 공식 리플릿을 웹 환경에 맞게 확장한 **밝은 청록/민트/화이트 기반 공공서비스형 디자인**입니다.

세부 기준은 `docs/design-tone-and-manner.md`를 따릅니다.

## 개발

```bash
npm install
npm run dev
npm run build
```
