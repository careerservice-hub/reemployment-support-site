# 재취업지원서비스 시행지원 홈페이지 디자인 톤앤매너

## 1. 기준 방향

이 홈페이지의 기준 톤앤매너는 **커리어플래닝서비스 공식 리플릿을 웹 환경에 맞게 확장한 밝은 공공서비스형 디자인**으로 한다.

핵심 인상은 다음과 같다.

- 정부지원사업 안내물의 신뢰감
- 케이잡스 수행기관의 실무적 전문성
- 중장년 재직자와 기업 담당자가 부담 없이 이해할 수 있는 밝고 정돈된 화면
- 과도한 홍보물 느낌보다 신청·상담·문의로 이어지는 행정적 명확성

## 2. 시각 키워드

- 밝음
- 청록/민트 중심
- 화이트 카드
- 연두·노랑 포인트
- 둥근 모서리
- 부드러운 그림자
- 공공사업 안내 리플릿 느낌
- 전문 홈페이지형 구성

피해야 할 방향:

- 다크 테크/스타트업 느낌
- 보라·마젠타 중심의 강한 그라데이션
- 블로그/게시판형 인상
- 과도한 애니메이션 또는 장식
- 장난감 같은 단순 벡터 이미지
- 지나치게 감성적인 생애설계 문구

## 3. 색상 기준

현재 CSS 기준 색상은 `src/styles/global.css`의 `:root`에 둔다.

- Primary Teal: `#008c95`
- Dark Teal: `#006d76`
- Soft Teal: `#e4f5f4`
- Mint: `#9edbd3`
- Mint Light: `#f1fbf8`
- Lime: `#c7de5a`
- Yellow: `#f6d94f`
- Navy Text: `#163b4a`
- Body Text: `#20323a`
- Muted Text: `#60727a`
- Paper Background: `#f6fbf9`

운영 원칙:

- 청록은 주 행동 버튼, 핵심 강조, 주요 시각 영역에 사용한다.
- 화이트 카드는 정보 묶음과 섹션 구분에 사용한다.
- 연두·노랑은 리플릿 연결감을 주는 보조 포인트로만 사용한다.
- 배경은 밝은 민트/화이트 계열을 유지한다.

## 4. 레이아웃 기준

홈 첫 화면:

- 좌측: 서비스명, 핵심 문장, 설명, CTA 버튼
- 우측: 리플릿 느낌의 추상 카드/상담 흐름 시각화
- 첫 CTA는 `커리어플래닝서비스 보기`, 보조 CTA는 `기업컨설팅 보기`

본문 섹션:

- 2분할 카드: 기업컨설팅 / 커리어플래닝서비스
- 4분할 카드: 대상 확인, 경력진단, 맞춤 설계, 실행 지원
- 마지막 CTA: 문의하기

카드 스타일:

- 흰색 또는 매우 밝은 민트 배경
- 28~34px 수준의 둥근 모서리
- 얕은 청록 그림자
- 상단에 청록-연두-노랑 얇은 포인트 라인

## 5. 문체 기준

공개 홈페이지 문체는 **수행기관 케이잡스 관점의 전문 행정 문체**로 한다.

좋은 문장:

- “중장년 재직자의 역량 향상을 위한 경력설계 컨설팅”
- “기업과 중장년 재직자가 함께 성장할 수 있도록 지원합니다.”
- “경력진단에서 실행계획까지 단계적으로 지원합니다.”
- “정부지원사업 기준에 맞춰 상담 진행, 결과 정리, 후속 안내를 지원합니다.”

피할 문장:

- 노사발전재단 또는 고용노동부가 직접 말하는 듯한 문장
- 공고문을 그대로 옮긴 듯한 딱딱한 행정 문장
- “인생 후반전”, “제2의 삶”처럼 일·직무 중심성이 약한 표현
- 아직 확정되지 않은 프로그램을 확정된 것처럼 보이게 하는 문장

## 6. 콘텐츠 관점

홈페이지는 다음 질문에 바로 답해야 한다.

1. 이 서비스가 무엇인가?
2. 누가 받을 수 있는가?
3. 기업과 개인에게 어떤 효용이 있는가?
4. 어떤 흐름으로 진행되는가?
5. 상담 또는 신청을 위해 무엇을 하면 되는가?

콘텐츠 우선순위:

1. 커리어플래닝서비스
2. 기업컨설팅
3. 진행 절차
4. K·JOBS 지원 방식
5. 문의

## 7. 향후 수정 시 적용 규칙

- 새 페이지를 만들 때도 이 문서의 색상·카드·문체 기준을 우선 적용한다.
- 리플릿 원문은 기준 자료로 사용하되, 웹 문구는 신청자와 기업 담당자가 이해하기 쉽게 재구성한다.
- 로고가 확정되면 헤더 텍스트 로고를 교체하되, 서비스명 중심 구조는 유지한다.
- Cloudflare Pages 또는 도메인 연결 후에도 톤앤매너는 동일하게 유지한다.
