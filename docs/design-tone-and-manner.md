# 재취업지원서비스 시행지원 홈페이지 디자인 톤앤매너

## 1. 기준 방향

이 홈페이지의 기준 톤앤매너는 **케이잡스 디자인 시스템을 웹 환경에 맞게 적용한 밝은 공공서비스형 디자인**으로 한다.

핵심 인상은 다음과 같다.

- 정부지원사업 안내물의 신뢰감
- 케이잡스 수행기관의 실무적 전문성
- 중장년 재직자와 기업 담당자가 부담 없이 이해할 수 있는 밝고 정돈된 화면
- 과도한 홍보물 느낌보다 신청, 상담, 문의로 이어지는 행정적 명확성

## 2. 시각 키워드

- K·JOBS 딥블루 브랜드 앵커
- 스카이블루 정보 강조
- 밝은 민트/화이트 공공서비스 배경
- 화이트 카드
- 절제된 연두·노랑 보조 포인트
- 8px~16px 중심의 단정한 모서리
- 얕은 블루 그림자
- 전문 홈페이지형 구성

피해야 할 방향:

- 다크 테크/스타트업 느낌
- 보라·마젠타 중심의 강한 그라데이션
- 블로그/게시판형 인상
- 과도한 애니메이션 또는 장식
- 장난감 같은 단순 벡터 이미지
- 지나치게 감성적인 생애설계 문구
- 근거 없는 통계, 성과 수치, 혜택 단정

## 3. 색상 기준

현재 CSS 기준 색상은 `src/styles/global.css`의 `:root`에 둔다.

- K·JOBS Deep Blue: `#003DA5`
- K·JOBS Deep Blue Dark: `#002B75`
- K·JOBS Blue Light: `#1A5CC8`
- K·JOBS Sky Blue: `#0099E6`
- Primary Teal: `#008C95`
- Dark Teal: `#006D76`
- Soft Teal: `#E4F5F4`
- Mint Light: `#F1FBF8`
- Lime: `#C7DE5A`
- Yellow: `#F6D94F`
- Neutral Light Gray: `#F4F6FA`
- Divider: `#D0D8E8`
- Body Text: `#1A1F2E`
- Muted Text: `#5A6480`

운영 원칙:

- 딥블루는 로고, 헤더, 주요 CTA, 핵심 강조에 사용한다.
- 스카이블루는 보조 강조, 정보 배지, 링크 상태에 사용한다.
- 민트/화이트 배경은 기존 공공서비스 톤을 유지하는 기반으로 사용한다.
- 연두·노랑은 리플릿 연결감을 주는 보조 포인트로만 제한한다.
- 장식 목적의 강한 그라데이션은 사용하지 않는다.

## 4. 레이아웃 기준

홈 첫 화면:

- 좌측: 서비스명, 핵심 문장, 설명, CTA 버튼
- 우측: 케이잡스 수행기관 관점의 절차 카드 시각화
- 첫 CTA는 `기업컨설팅 확인`, 보조 CTA는 `커리어플래닝서비스 보기`

본문 섹션:

- 3분할 요약: 재취업지원서비스 시행지원 / 기업컨설팅 / 커리어플래닝서비스
- 2분할 카드: 기업컨설팅 / 커리어플래닝서비스
- 4분할 카드: 대상 확인, 운영 설계, 경력 상담, 결과 정리
- 마지막 CTA: 문의하기

카드 스타일:

- 흰색 또는 매우 밝은 회청색 배경
- 8px~16px 중심의 단정한 모서리
- 얕은 딥블루 그림자
- 강조 카드에는 상단 또는 좌측에 딥블루 포인트 라인을 사용한다.

## 5. 문체 기준

공개 홈페이지 문체는 **수행기관 케이잡스 관점의 전문 행정 문체**로 한다.

좋은 문장:

- “재취업지원서비스를 기업 현장에서 실행 가능하게 정리합니다.”
- “기업 담당자는 절차와 준비사항을 명확히 확인할 수 있습니다.”
- “경력진단에서 실행계획까지 단계적으로 지원합니다.”
- “정부지원사업 기준에 맞춰 상담 진행, 결과 정리, 후속 안내를 지원합니다.”

피할 문장:

- 노사발전재단 또는 고용노동부가 직접 말하는 듯한 문장
- 공고문을 그대로 옮긴 듯한 딱딱한 행정 문장
- “인생 후반전”, “제2의 삶”처럼 일·직무 중심성이 약한 표현
- 아직 확정되지 않은 프로그램을 확정된 것처럼 보이게 하는 문장
- 이모지 또는 장식적 감탄 표현

## 6. 콘텐츠 관점

홈페이지는 다음 질문에 바로 답해야 한다.

1. 이 서비스가 무엇인가?
2. 누가 받을 수 있는가?
3. 기업과 개인에게 어떤 효용이 있는가?
4. 어떤 흐름으로 진행되는가?
5. 상담 또는 신청을 위해 무엇을 하면 되는가?

콘텐츠 우선순위:

1. 재취업지원서비스 시행지원
2. 기업컨설팅
3. 커리어플래닝서비스
4. 진행 절차
5. K·JOBS 지원 방식
6. 문의

## 7. 향후 수정 시 적용 규칙

- 새 페이지를 만들 때도 이 문서의 색상, 카드, 문체 기준을 우선 적용한다.
- 리플릿 원문은 기준 자료로 사용하되, 웹 문구는 신청자와 기업 담당자가 이해하기 쉽게 재구성한다.
- 로고는 `public/assets/kjobs-logo.svg`를 사용한다.
- 문의 이메일과 대표 연락처가 확정되기 전까지는 `추후 확정` 표시를 유지한다.
- GitHub Pages 배포 설정은 별도 승인 없이 변경하지 않는다.
